SQL Project 1 - Exploring Customer Behavior Through Database Analysis

Project Overview:

This SQL project analyzes customer behavior for an online food delivery platform using relational database analysis. The goal of the project is to understand customer purchasing patterns, product popularity, and the effectiveness of the loyalty program using SQL queries.

The analysis was performed on multiple relational tables containing customer transactions, product details, and membership information. By exploring customer behavior patterns, the project provides actionable insights for improving customer retention, product marketing, and revenue growth.

Business Problem:

Online food delivery platforms generate large volumes of transactional data. Understanding how customers interact with products and loyalty programs can help businesses:

Improve customer retention
Optimize menu offerings
Increase loyalty program adoption
Personalize product recommendations

This project analyzes customer transaction data to uncover patterns in purchasing behavior and membership impact.

Dataset Description:

The project dataset contains 5 relational tables:

Table	              Description
Sales	              Records of orders placed by customers
Product	            Menu items with product names and prices
Users	              Customer signup details
User_Name	          Mapping of user IDs to customer names
Golduser_Signup	    Records of customers who upgraded to gold membership

These tables were linked using primary and foreign key relationships to perform multi-table analysis.

SQL Project 2 - Workforce Income Analysis Using SQL

Project Overview:

This SQL project analyzes global workforce salary trends using structured data stored in a relational database. The project focuses on understanding salary distribution across job roles, experience levels, company sizes, and geographic regions.

The dataset contains workforce records from 2020 to 2024, providing insights into salary growth trends, remote work adoption, and job market patterns in the data and technology industry.

Business Problem:

Organizations need accurate salary insights to:

Benchmark salaries across roles and regions
Improve hiring strategies
Understand remote work trends
Retain employees through competitive compensation

This project uses SQL to analyze workforce salary data and identify market trends.

Dataset Overview:

The dataset includes 13,972 workforce records containing employee salary and employment details.

Key fields include:

Column	             Description
work_year	           Year of employment
experience_level	   Employee seniority level (EN, MI, SE, EX)
employment_type	     Full-time, Part-time, Contract
job_title	           Employee job role
salary	             Reported salary
salary_in_usd	       Standardized salary value
employee_residence	 Employee location
remote_ratio	       Remote work percentage
company_location	   Employer location
company_size	       Size of organization

The data was imported into SQL Server from a CSV dataset and analyzed using SQL queries.

Key Insights:

Workforce Distribution

Most employees work in large companies
Major employment hubs include:
United States
India
Germany
Canada
United Kingdom

Salary Trends

Entry and mid-level roles experienced the highest salary growth.
Senior-level salaries vary significantly across countries.

Example:

Country	     Senior Level Salary
Israel	     $266K
Turkey	     $18K

This highlights large geographic salary gaps.

Remote Work Trends:

Nearly 75% of fully remote employees earn more than $100K.
Remote work adoption has declined from 2021 to 2024.

Job Role Insights:

High growth roles include:

Machine Learning Engineer
Data Science Consultant
Applied Scientist

Career mobility between roles can significantly increase salary.

Example:
Data Analyst → Applied Scientist salary increase of approximately $79K.

Business Recommendations:

Salary Benchmarking Strategy

Organizations should implement region-specific salary bands due to major geographic salary differences.

Talent Acquisition:

Focus hiring in high-paying mid-level markets such as:

Qatar
Australia
Canada
United States

Remote Work Policy:

Remote employees command higher salaries; companies should balance flexibility and compensation.

Career Development Programs:

Encourage employees to transition into high-growth roles such as ML Engineer and Data Scientist.
